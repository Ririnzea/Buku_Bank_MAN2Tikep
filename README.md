# Aplikasi Buku Pembantu Bank - MAN 2 Kota Tidore Kepulauan

Aplikasi web sederhana untuk mengelola buku pembantu bank sekolah.

## Fitur

- Menampilkan tabel transaksi bank dengan kolom: Nomor, Tanggal, No Bukti, Uraian, Kode Rekening, Penerimaan, Pengeluaran, dan Saldo
- Tambah transaksi baru
- Edit transaksi yang sudah ada
- Hapus transaksi
- Perhitungan saldo otomatis
- Format mata uang Indonesia
- Fungsi cetak
- Data contoh sesuai referensi

## Cara Menggunakan

1. Buka file `index.html` di browser
2. Klik "Tambah Transaksi" untuk menambah data baru
3. Klik "Edit" pada baris transaksi untuk mengubah data
4. Klik "Hapus" untuk menghapus transaksi
5. Klik "Cetak" untuk mencetak dokumen

## Teknologi

- HTML5
- CSS3
- JavaScript (Vanilla)

## Catatan

Aplikasi ini menggunakan localStorage browser untuk menyimpan data. Data akan tersimpan selama browser tidak dihapus cache-nya.
